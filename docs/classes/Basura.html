  • Create some Basura.

          arrayLength: number;

          noBoxed: boolean;
          output: boolean;
          randBytes: BasuraRandBytes;
          scripts: string[];
          stringLength: number;
          types: Record<string, BasuraGenerator>;
      } = {}

      Options.

      • arrayLength: number

        Maximum size of arrays and objects.

      • cborSafe: boolean

        Generate only CBOR-safe types?

      • depth: number

        Maximum depth of object to create.

      • edgeFreq: number

        How often do edge cases happen (0-1)?

      • jsonSafe: boolean

        Generate only JSON-safe types?

      • noBoxed: boolean

        Ignore boxed types, like String?

      • output: boolean

        Add custom inspect functions that make output parseable JS?

      • randBytes: BasuraRandBytes

        Randomness source. Defaults to a thin wrapper around crypto.randomBytes.

      • scripts: string[]

        List of script names to select from for generating strings. Defaults to all Unicode scripts from data.json.

      • stringLength: number

        Maximum size of generated strings. (in codepoints), BigInts (in bytes), and buffers (in bytes).

      • types: Record<string, BasuraGenerator>

        Additional types. The key is the type name, the value is a function used to generate, or null to prevent that type from being generated.

    Returns Basura

Static quoteSymbols

  • quoteSymbols(str): string

  • Symbols can't take a custom inspect function. They always output Symbol(foo). This function tries to find all of those in a string, and replace them with Symbol.for('foo'), which will create a valid symbol when parsed as JavaScript.

    This function currently fails on edge cases such as what is created from Symbol(')))'), which luckily shouldn't be created from this library.

    Parameters

    • str: string

      The string to modify.
